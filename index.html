機能

必要なものだけ、余計なものは何もない。

⊞

レイアウトを保存

メニューバーからワンクリックで、開いているすべてのウィンドウの位置とサイズを記録します。

↩

瞬時に適用

保存したレイアウトを選ぶだけで、ウィンドウが元の位置に自動で移動します。

◫

重なり順も再現

位置・サイズだけでなく、ウィンドウの前後の順序（Z オーダー）も保存・再現します。

⌘

キーボードショートカット

⌘⇧S を押すだけで、マウスを使わずに現在の配置を保存できます。

◑

マルチディスプレイ対応

どのディスプレイにあるウィンドウかを記録するため、複数画面の環境でも正確に再現します。

↑

メニューバーに常駐

Dock アイコンなし。邪魔しません。必要なときだけメニューバーから呼び出せます。

使い方

はじめから終わりまで。

01

インストール

DMG ファイルをダウンロードして開き、Displave をアプリケーションフォルダへドラッグします。起動すると、メニューバーにアイコンが表示されます。

ℹ 初回起動時にアクセシビリティ権限の付与を求めるダイアログが表示されます。「システム設定を開く」をクリックして権限を有効にしてください。
02

アクセシビリティ権限を付与する

Displave はウィンドウの位置・サイズを取得・操作するためにアクセシビリティ権限が必要です。

設定手順

システム設定 → プライバシーとセキュリティ → アクセシビリティ → Displave をオンにする

収集するデータ

ウィンドウの位置・サイズ・アプリ名のみ。個人データや画面の内容は一切取得しません。

03

現在のレイアウトを保存する

ウィンドウを好みの配置に並べたら、メニューバーの Displave アイコンをクリックして Save Current Layout を選択します。または ⌘⇧S でも保存できます。

名前を入力して保存すると、以降のメニューに表示されます。レイアウトはいくつでも保存できます。

💡 「開発モード」「ミーティング」「執筆」など、作業内容がわかる名前をつけると切り替えがスムーズです。
04

レイアウトを適用する

メニューバーの Displave アイコンをクリックし、保存済みのレイアウト名にカーソルを合わせて Apply をクリックします。

アプリが起動していない場合は自動で起動し、すべてのウィンドウが保存時の位置・サイズに戻ります。

05

レイアウトを削除する

不要になったレイアウトは、メニューバーからレイアウト名にカーソルを合わせて Delete をクリックすると削除できます。確認ダイアログが表示されます。

FAQ

よくある質問

アプリを閉じてもレイアウトは保存されますか？

はい。レイアウトはディスク上に保存されるため、Displave を再起動しても引き続き利用できます。

適用時にアプリが起動していない場合はどうなりますか？

Displave が自動でアプリを起動し、起動後にウィンドウの位置を設定します。

Slack などタイトルが変わるアプリでも正しく動きますか？

はい。タイトルではなくアプリ名を基準にウィンドウを特定するため、動的にタイトルが変わるアプリでも問題なく動作します。

Intel Mac でも使えますか？

現在のビルドは Apple Silicon (arm64) 向けです。Intel 版が必要な場合は GitHub からソースをビルドしてください。

表示言語を変えるには？

Displave は macOS のシステム言語設定を自動で読み取ります。日本語に設定されている場合は日本語、それ以外は英語で表示されます。
