使い方

はじめから終わりまで。

01

インストール

DMG をダウンロードして開き、Displave をアプリケーションフォルダへドラッグします。起動するとメニューバーにアイコンが表示されます。

ℹ初回起動時にアクセシビリティ権限の付与を求めるダイアログが表示されます。「システム設定を開く」をクリックして権限を有効にしてください。
02

アクセシビリティ権限を付与する

Displave はウィンドウの位置・サイズを取得・操作するためにアクセシビリティ権限が必要です。

設定手順

システム設定 → プライバシーとセキュリティ → アクセシビリティ → Displave をオンにする

収集するデータ

ウィンドウの位置・サイズ・アプリ名のみ。個人データは一切取得しません。

03

レイアウトを保存する

ウィンドウを好みの配置に並べたら、メニューバーの Displave アイコンをクリックして Save Layout を選択。または ⌘⇧S でも保存できます。

💡「開発モード」「ミーティング」など、作業内容がわかる名前をつけると切り替えがスムーズです。
04

レイアウトを適用する

メニューバーのアイコンをクリックし、保存済みのレイアウト名にカーソルを合わせて Apply をクリック。アプリが起動していない場合は自動で起動します。

05

レイアウトを削除する

レイアウト名にカーソルを合わせて Delete をクリック。確認ダイアログが表示されます。

FAQ

よくある質問

アプリを閉じてもレイアウトは保存されますか？

はい。レイアウトはディスクに保存されるため、再起動後も引き続き利用できます。

適用時にアプリが起動していない場合は？

Displave が自動でアプリを起動し、起動後にウィンドウの位置を設定します。

Slack などタイトルが変わるアプリでも動きますか？

はい。アプリ名を基準にウィンドウを特定するため、動的なタイトル変化は問題ありません。

Intel Mac でも使えますか？

現在のビルドは Apple Silicon 向けです。Intel 版が必要な場合は GitHub からソースをビルドしてください。

表示言語を変えるには？

macOS のシステム言語設定を自動で読み取ります。日本語設定なら日本語、それ以外は英語で表示されます。
