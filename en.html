Usage

From start to finish.

01

Install

Download the DMG, open it, and drag Displave to your Applications folder. Launch it — a Displave icon will appear in your menu bar.

ℹ On first launch, a dialog will ask you to grant Accessibility permission. Click "Open System Settings" and enable Displave in the list.
02

Grant Accessibility Permission

Displave needs Accessibility permission to read and move window positions.

How to enable

System Settings → Privacy & Security → Accessibility → toggle Displave on

What is collected

Only window position, size, and app name. No screen content or personal data is ever read.

03

Save the current layout

Arrange your windows the way you like, then click the Displave icon in the menu bar and choose Save Current Layout. You can also press ⌘⇧S.

Enter a name and save. Repeat for as many layouts as you need.

💡 Use descriptive names like "Coding", "Meeting", or "Writing" so you can switch modes at a glance.
04

Apply a layout

Click the Displave icon in the menu bar, hover over a saved layout name, and click Apply.

If an app isn't running, Displave launches it automatically before positioning the windows.

05

Delete a layout

Hover over a layout name in the menu and click Delete. A confirmation dialog will appear before anything is removed.

FAQ

Common questions.

Are layouts saved after I quit Displave?

Yes. Layouts are persisted to disk, so they're available again the next time you launch Displave.

What happens if an app isn't open when I apply a layout?

Displave launches the app automatically, waits for it to be ready, then positions its windows.

Does it work with apps like Slack that change their window title dynamically?

Yes. Displave identifies windows by app name rather than window title, so dynamic titles aren't a problem.

Is an Intel build available?

The current release is Apple Silicon (arm64) only. For Intel, clone the repository and build from source.

How do I change the display language?

Displave reads your macOS system language automatically. It displays in Japanese when the system language is set to Japanese, and in English otherwise.
