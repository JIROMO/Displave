Usage

From start to finish.

01

Install

Download the DMG, open it, and drag Displave to Applications. A Displave icon will appear in your menu bar.

ℹOn first launch, a dialog will ask you to grant Accessibility permission. Click "Open System Settings" and enable Displave.
02

Grant Accessibility Permission

Displave needs Accessibility permission to read and move window positions.

How to enable

System Settings → Privacy & Security → Accessibility → toggle Displave on

What is collected

Only window position, size, and app name. No screen content or personal data.

03

Save the current layout

Arrange your windows, then click the Displave icon → Save Layout. Or press ⌘⇧S.

💡Use descriptive names like "Coding", "Meeting", or "Writing" for easy switching.
04

Apply a layout

Click the menu bar icon, hover over a saved layout, and click Apply. If an app isn't running, Displave launches it automatically.

05

Delete a layout

Hover over a layout and click Delete. A confirmation dialog will appear.

FAQ

Common questions.

Are layouts saved after I quit Displave?

Yes. Layouts are persisted to disk and available again the next time you launch.

What if an app isn't open when I apply a layout?

Displave launches it automatically, then positions its windows.

Does it work with apps like Slack that change their title?

Yes. Displave identifies windows by app name, so dynamic titles aren't a problem.

Is an Intel build available?

The current release is Apple Silicon only. Clone the repo and build from source for Intel.

How do I change the display language?

Displave reads your macOS system language automatically — Japanese or English.
