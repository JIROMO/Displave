How it works

Three steps,
done.

01

Arrange your windows

Set up your workspace exactly how you like it — coding layout, meeting layout, writing layout. Anything goes.

02

Save Current Layout

Click the Displave icon in the menu bar and choose Save Current Layout, or press ⌘⇧S. Give it a name and save.

03

Apply whenever you need it

Click the menu bar icon, select a saved layout, and click Apply. Your windows move back into place automatically.

Documentation

Get started quickly.

Installation

Download the DMG, drag Displave to Applications, and launch it. Grant accessibility permission when prompted.

Permissions

Displave requires Accessibility permission to read and move window positions. No personal data is collected.

Keyboard Shortcut

Press ⌘⇧S to save the current layout at any time without opening the menu.

Language

Displave automatically switches between English and Japanese based on your macOS language setting.
